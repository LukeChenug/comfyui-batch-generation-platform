# 🎨 ComfyUI批量生图平台

> 专为儿童绘本智能体项目打造的AI图像批量生成平台

## 🚀 快速启动

```bash
# 一键启动平台
./quick_start.sh

# 访问管理界面
open batch_generation_dashboard.html
```

## 📁 文件结构

```
绘本自动化工具/
├── batch_generation_dashboard.html    # 🌐 批量管理界面
```

## ⚙️ 使用步骤

### 1. 配置ComfyUI服务器地址
在管理界面中设置您的ComfyUI服务器地址

### 2. 提交生图任务
- **单个任务**：输入提示词和参数
- **批量任务**：上传包含多个提示词的文本

### 3. 实时监控进度
通过WebSocket实时查看任务状态和进度

### 4. 下载生成结果
任务完成后可直接在界面中查看和下载图片

## 🔧 高级功能

- ✅ 任务队列管理

- ✅ 批量文本处理
- ✅ 图片自动保存
- ✅ 任务历史记录
- ✅ 错误自动重试

## 🆘 技术支持

如遇问题，请检查：
1. ComfyUI服务器是否正常运行
2. Python依赖是否正确安装
3. 查看 `logs/` 目录中的错误日志
